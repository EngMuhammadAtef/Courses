What is Java?

Java is a popular programming language, created in 1995.

What can Java do?

It is used for:
Mobile applications (specially Android apps)
Desktop applications
Web applications
Web servers and application servers
Games
Database connection

Why Use Java?

Java works on different platforms (Windows, Mac, Linux, Raspberry Pi, etc.)
Java is one of the most popular programming language in the world
Java is easy to learn and simple to use
Java is open-source and free.
Java is secure, fast and powerful.
Java is an object oriented language which gives a clear structure to programs and allows code to be reused, lowering development costs

we created a Java file called Main.java, and we used the following code to print "Hello World" to the screen:

public class Main {

Java Variables

Variables are containers for storing data values. A variable is created the moment you first assign a value to it.

Example

int x = 5;
String name = "John";
System.out.println(x);
System.out.println(name);

>> 5
>> John

Java Data Types

In programming, data type is an important concept.
Variables can store data of different types, and different types can do different things.
As explained in the previous chapter, a variable in Java must be a specified data type:

Text Type :===>>> str # String myText = "Hello";
Int Type :===>>>> int # int myNum = 5;
Float Types :===>>> float # float myFloatNum = 5.99f;
Char Types :===>>> char # char myLetter = 'D';
Boolean Type :===>>> bool # boolean myBool = true;

Java Conditions and If statements

Java supports the usual logical conditions from mathematics:
Equals: a == b
Not Equals: a != b
Less than: a < b
Less than or equal to: a <= b
Greater than: a > b
Greater than or equal to: a >= b

Example

int a = 10;
int b = 5;
if (b > a) {System.out.println("b is greater than a")}
else if (a == b) {System.out.println("a and b are equal")}
else {System.out.println("a is greater than b")}

>> a is greater than b

Java While Loop

Loops can execute a block of code as long as a specified condition is reached.
Loops are handy because they save time, reduce errors, and they make code more readable.
The while loop loops through a block of code as long as a specified condition is true:

Example

// Print i as long as i is less than 5:
int i = 1;
while (i < 5) {
System.out.println(i);
i++;
}

>> 1
>> 2
>> 3
>> 4

Java For Loop

When you know exactly how many times you want to loop through a block of code, use the for loop instead of a while loop:

Example

for (int i = 1; i < 5; i++) {
System.out.println(i);}

>> 1
>> 2
>> 3
>> 4

Java Methods

A Method is a block of code which only runs when it is called.
You can pass data, known as parameters, into a method.
Methods are used to perform certain actions, and they are also known as functions.
Why use methods? To reuse code: define the code once, and use it many times.

Create a Method

public class Main {
static void myMethod() {
System.out.println("I just got executed!");}
public static void main(String[] args) {
myMethod();}
}

>> I just got executed!

Java Arrays

Arrays are used to store multiple values in a single variable, instead of declaring separate variables for each value.
To declare an array, define the variable type with square brackets:

Example

// Create an array containing car names:
String[] cars = {"Volvo", "BMW", "Ford", "Mazda"};

Access the Elements of an Array


// You access an array element by referring to the index number.
String[] cars = {"Volvo", "BMW", "Ford", "Mazda"};
System.out.println(cars[0]);

>> Volvo

Modify the value of the first array item:


cars[0] = "Opel";
System.out.println(cars);

>> {"Opel", "BMW", "Ford", "Mazda"}
